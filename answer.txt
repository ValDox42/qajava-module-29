В изначальном коде были неверно заданы значения для isTeenager.
А именно: 
(age < 19) должно быть задано как (age <= 19). Из-за этого значение 19 не попадает в границы Boolean и выдает False.
Также необходимо было добавить нижнюю границу возраста (age >= 13). Без этого все значения возраста до 19 лет попадают под тинейджеров, что не верно, ибо имеется заданный диапазон от 13 до 19 лет.
Моё видение правильного кода: 
public class Person {
   public static boolean isTeenager(int age)
   {
       boolean result;
       if ((age >= 13) && (age <= 19)) {
           result = true;
       }
       else {
           result = false;
       }
       return result;
   }
}

Тесты подтверждают указанные ошибки.
